# DriveGuard-AI
An Intelligent Predictive Vehicle Maintenance System using Machine Learning


# Road Damage Detection Module

## Member
Member 4

## Objective

Detect potholes and road surface damages using YOLOv8 and provide maintenance recommendations for predictive vehicle health monitoring.

## Dataset

RDD2022 (Road Damage Detection Dataset)

## Model

YOLOv8 Nano

## Detected Classes

- D00 - Longitudinal Crack
- D10 - Transverse Crack
- D20 - Alligator Crack
- D40 - Pothole

## Workflow

Road Image
↓
YOLO Detection
↓
Road Damage Score
↓
Vehicle Stress Analysis
↓
Predictive Maintenance
↓
Dashboard

## Output

- Detected road damage
- Road Damage Score
- Maintenance Recommendation
